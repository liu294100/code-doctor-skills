### 3. 业务规则（根据功能自动填充）

以下是港美股常见功能的业务规则知识库，生成方案时按需引用：

#### 港股交易规则库
```
交易时段：
  开市前时段：09:00-09:30（竞价）
  持续交易：09:30-12:00, 13:00-16:00
  收市竞价：16:00-16:10

委托类型：
  增强限价单（ELO）：最多匹配10个价位，未成交部分转为限价盘
  特别限价单（SLO）：最多匹配1个价位，未成交部分撤销
  竞价单（AO）：仅竞价时段，以最终IEP成交
  竞价限价单（ALO）：竞价时段限价单

价位表（Spread Table）：
  0.01-0.25: 0.001    0.25-0.50: 0.005
  0.50-10: 0.010      10-20: 0.020
  20-100: 0.050       100-200: 0.100
  200-500: 0.200      500-1000: 0.500
  1000-2000: 1.000    2000-5000: 2.000
  5000+: 5.000

交收周期：T+2（CCASS）
最小交易单位：每手股数因股票而异
碎股：不足一手只能卖出，通过碎股市场
涨跌幅：无限制（有市场波动调节机制 VCM）
卖空：需借券，卖空价不低于最佳买入价（Uptick Rule）
```

#### 美股交易规则库
```
交易时段：
  盘前：04:00-09:30 ET（仅限价单）
  正常：09:30-16:00 ET（所有类型）
  盘后：16:00-20:00 ET（仅限价单）
  注意夏令时/冬令时

委托类型：
  市价单（MKT）：正常时段
  限价单（LMT）：所有时段
  止损单（STP）：触发后变为市价单
  止损限价单（STP LMT）：触发后变为限价单
  MOO/MOC/LOO/LOC：开盘/收盘特殊订单

交收周期：T+1（DTCC）
最小交易单位：1股（支持碎股 0.0001 股）
最小变动价位：$0.01（>=$1）/ $0.0001（<$1）
涨跌幅：无固定限制，有 LULD 机制和熔断机制
PDT 规则：保证金账户 5 日内日内交易 >= 4 次需 $25,000
卖空：需借券，有 SSR（Short Sale Restriction）规则
```

#### 港股通（北向）规则库
```
可交易标的：沪股通/深股通标的名单
交易时段：跟随 A 股（09:15-15:00）
交收周期：T+1
最小单位：100股（科创板/创业板可1股）
涨跌幅：主板±10%，科创/创业±20%，北交所±30%
T+0：北向投资者当日买入可当日卖出
额度：每日520亿人民币
汇率：参考汇率（冻结用）+ 结算汇率（交收用）
不支持：融资融券、大宗交易
```

## 港美股常见功能方案要点速查

### 股票交易（买入/卖出）
- 核心链路：参数校验 → 风控 → 冻结资金/持仓 → 报盘 → 回报处理
- 关键点：幂等、状态机、资金冻结事务、回报去重补全

### 改单/撤单
- 港股：支持原地改单（价格/数量），交易所直接修改
- 美股：Cancel-Replace 模式（撤旧下新），需处理中间状态
- 关键点：并发处理（撤单和成交同时到达）、部分成交后改单

### 条件单（止损/止盈/追踪止损）
- 行情驱动评估，每个 Tick 检查
- 持久化 + 服务重启恢复
- GTC 跨交易日有效
- 触发后转普通委托，废单需通知

### 暗盘交易（港股 IPO）
- 独立交易通道，非交易所撮合
- 交易时段：上市前一天 16:15-18:30
- 仅支持限价单
- 成交后 T+2 交收

### 碎股交易
- 港股：碎股只能卖出，走碎股市场
- 美股：碎股可买可卖，券商内部撮合
- 碎股持仓独立展示

### 盘前盘后交易（美股）
- 仅支持限价单
- 流动性低，价差大，需提示风险
- 盘前盘后成交不计入当日成交量

### 融券卖空
- 需要借券（券源查询）
- 港股 Uptick Rule：卖空价 >= 最佳买入价
- 美股 SSR：触发后只能在上涨时卖空
- 借券费用：按日计息
- 强制买回：券源召回时必须买回

### 大宗交易/协议交易
- 港股：大手交易（Block Trade），场外协商价格
- 美股：Block Trade，通常通过暗池执行
- 最低数量/金额门槛
- 报价偏离限制
